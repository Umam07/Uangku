# 💸 Uangku

**Uangku** adalah aplikasi pencatatan keuangan pribadi harian yang dirancang dengan antarmuka yang sangat menarik, interaktif, dan modern. Aplikasi ini menggunakan estetika **Apple Glass / Glassmorphism** yang premium, responsif terhadap pergantian tema (Light & Dark), serta dilengkapi visualisasi chart yang kaya untuk melacak pengeluaran harian Anda.

---

## ✨ Fitur Unggulan

- 🏠 **Dashboard (Beranda)**: Kartu saldo interaktif dengan visualisasi pie chart distribusi pengeluaran secara real-time.
- 🕒 **Riwayat Keuangan**: Pencarian transaksi instan dan filter kategori, dilengkapi dengan fitur swipe-to-delete dengan opsi undo.
- ➕ **Input Transaksi & Scan Struk**: Pencatatan manual yang ringkas dan simulator **Scan Struk Belanja (OCR)** yang dilengkapi dengan camera frame viewfinder dan animasi laser scanner.
- 📊 **Laporan & Analisis**: Perbandingan pemasukan vs pengeluaran mingguan/bulanan menggunakan dual Bar Chart dan Pie Chart yang interaktif.
- ⚙️ **Setelan & Profil**: Kustomisasi profil, data-reset ke status default, dan **Segmented Theme Selector** untuk mengubah tema (Terang, Gelap, Sistem) secara instan.

---

## 🛠️ Tech Stack & Dependencies

- **Framework**: [Flutter](https://flutter.dev) (Dart)
- **State Management**: `ValueNotifier` & `StatefulWidget`
- **Charts**: `fl_chart`
- **Fonts**: `google_fonts` (Poppins)
- **Local Persistence**: `shared_preferences`
- **Icons**: Material Icons & `cupertino_icons`

## 🚀 Cara Memulai

### Prasyarat
Pastikan Anda telah menginstal Flutter SDK versi terbaru di komputer Anda.

### Instalasi & Menjalankan Aplikasi

1. Clone repositori ini:
   ```bash
   git clone https://github.com/Umam07/Uangku.git
   ```
2. Pindah ke direktori project:
   ```bash
   cd Uangku
   ```
3. Unduh dependencies:
   ```bash
   flutter pub get
   ```
4. Jalankan aplikasi pada emulator atau perangkat fisik yang terhubung:
   ```bash
   flutter run
   ```
